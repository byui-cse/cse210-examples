# MyFirstMauiApp

This is a .NET MAUI project created for .NET 10.0.

## Prerequisites

To work with this project, you need to have the MAUI workload installed.

### Install MAUI Workload
Run the following command in your terminal:
```bash
dotnet workload install maui
```

### Create a New MAUI Project
To create a new project from scratch, use:
```bash
dotnet new maui -n MyFirstMauiApp
```

## Running the Application

You can run the application from the command line using the following command:

```bash
dotnet run --framework net10.0-windows10.0.19041.0
```

*Note: You may need to adjust the framework version based on your target platform (e.g., Android, iOS, MacCatalyst).*
