Brother Crosby's In Class Exercises and Examples for CSE 210

These examples may be incomplete, demonstrations of what not to do, demonstrations of what to do, and fully or partially AI generated.

# Prerequisites

.NET 10.0

# Usage

All of these projects can be run in the terminal with the dotnet command:

```bash
cd Encapsulation/Bookcase
dotnet run
```
